#  Detection of cognitive and language impairment using voice data
음성 데이터를 이용한 인지•언어 기능 저하 감지


## 💻 프로젝트 설명
- LLM을 활용하여 노인분들과 대화를 유도하고 수집한 음성 데이터를 발화 특성과 함께 추출한다. 
AI 모델 학습, 예측을 통하여 인지•언어 기능을 점수화하여 사용자에게 나타낸다.
